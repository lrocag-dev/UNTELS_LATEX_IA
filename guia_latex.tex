	\section{Introducción}
	
	\lettrine{E}{l} presente documento es una guía práctica para estudiantes de Ingeniería Electrónica que desean aprovechar el poder de \textbf{GitHub Copilot} para crear documentos \textbf{LaTeX} de alta calidad utilizando \textbf{Visual Studio Code}.
	
	GitHub Copilot es un asistente de programación impulsado por Inteligencia Artificial que \textbf{autocompleta código en tiempo real}. Funciona como un "programador en pareja" que sugiere líneas completas, fragmentos de código e incluso párrafos enteros basándose en el contexto de tu documento.
	
	\subsection*{¿Qué aprenderás en esta guía?}
	\begin{itemize}
		\item Instalar y configurar VS Code con LaTeX
		\item Activar y configurar GitHub Copilot
		\item Usar Copilot para generar ecuaciones, tablas y diagramas
		\item Mejores prácticas y trucos avanzados
		\item Solución de problemas comunes
	\end{itemize}

	\subsection{Paso 1: Instalar Visual Studio Code}
	
	\begin{enumerate}
		\item Descargar VS Code desde \href{https://code.visualstudio.com}{code.visualstudio.com}
		\item Ejecutar el instalador \codigo{VSCodeUserSetup-x64-xxx.exe}
		\item \textbf{Importante:} Marcar las siguientes opciones durante la instalación:
		\begin{itemize}
			\item \checkmark Add "Open with Code" action to Windows Explorer file context menu
			\item \checkmark Add "Open with Code" action to Windows Explorer directory context menu
			\item \checkmark Register Code as an editor for supported file types
			\item \checkmark Add to PATH (requiere reiniciar)
		\end{itemize}
	\end{enumerate}
	
	\subsection{Paso 2: Instalar MiKTeX (LaTeX)}
	
	\begin{enumerate}
		\item Descargar MiKTeX desde \href{https://miktex.org}{miktex.org}
		\item Ejecutar el instalador \codigo{basic-miktex-xxx.exe}
		\item Elegir \textbf{"Install for all users"}
		\item En "Install missing packages on the fly" seleccionar \textbf{"Yes"}
		\item Completar la instalación y cerrar
	\end{enumerate}
	
	\subsection{Paso 3: Instalar extensiones de VS Code}
	
	Abrir VS Code y presionar \comando{Ctrl+Shift+X} para abrir el mercado de extensiones. Instalar las siguientes:
	
	\begin{table}[H]
		\centering
		\begin{tabular}{@{}lll@{}}
			\toprule
			\textbf{Extensión} & \textbf{ID/Publicador} & \textbf{Función} \\
			\midrule
			LaTeX Workshop & James-Yu.latex-workshop & Compilación y previsualización \\
			GitHub Copilot & GitHub.copilot & Autocompletado con IA \\
			GitHub Copilot Chat & GitHub.copilot-chat & Chat interactivo con IA \\
			GitLens & eamodio.gitlens & Gestión avanzada de Git \\
			\bottomrule
		\end{tabular}
		\caption{Extensiones necesarias}
		\label{tab:extensiones}
	\end{table}
	
	\subsubsection*{Instalación paso a paso:}
	\begin{enumerate}
		\item En la barra de búsqueda, escribir "LaTeX Workshop"
		\item Click en "Install"
		\item Repetir para "GitHub Copilot"
		\item Repetir para "GitLens"
	\end{enumerate}
	
	\subsection{Paso 4: Configurar GitHub Copilot}
	
	\subsubsection*{Para estudiantes (GRATIS):}
	\begin{enumerate}
		\item Ir a \href{https://education.github.com}{education.github.com}
		\item Verificar estado de estudiante con correo institucional (\codigo{@untels.edu.pe})
		\item Solicitar \textbf{GitHub Copilot Student Pack}
	\end{enumerate}
	
	\subsubsection*{Para docentes:}
	\begin{enumerate}
		\item Ir a GitHub → Settings → Copilot
		\item Seleccionar plan: \$10 USD/mes o \$100 USD/año
	\end{enumerate}
	
	\subsubsection*{Activar Copilot en VS Code:}
	\begin{enumerate}
		\item Abrir VS Code
		\item Presionar \comando{Ctrl+Shift+P}
		\item Escribir: \codigo{GitHub Copilot: Sign in}
		\item Seguir instrucciones en el navegador
		\item Verificar que el ícono de Copilot aparece en la barra inferior (\faCheck)
	\end{enumerate}
	
	\begin{nota}
Si el ícono aparece con una línea diagonal, significa que está deshabilitado. Click en él y selecciona \textbf{Enable Completions}.
	\end{nota}

	\section{Configuración Óptima}
	
	\subsection{Configuración de LaTeX Workshop}
	
	Abrir la paleta de comandos (\comando{Ctrl+Shift+P}) y seleccionar \codigo{Preferences: Open Settings (JSON)}. Agregar la siguiente configuración:
	
	\begin{lstlisting}[caption=Configuración de settings.json]
		{
			// Configuración de LaTeX Workshop
			"latex-workshop.latex.autoBuild.run": "onSave",
			"latex-workshop.latex.outDir": "build",
			"latex-workshop.latex.clean.enabled": true,
			
			// Herramientas de compilación
			"latex-workshop.latex.tools": [
			{
				"name": "latexmk",
				"command": "latexmk",
				"args": [
				"-synctex=1",
				"-interaction=nonstopmode",
				"-file-line-error",
				"-pdf",
				"-outdir=build",
				"%DOC%"
				]
			}
			],
			
			// Recetas de compilación
			"latex-workshop.latex.recipes": [
			{
				"name": "latexmk",
				"tools": ["latexmk"]
			}
			],
			
			// Habilitar Copilot para todos los lenguajes
			"github.copilot.enable": {
				"*": true,
				"plaintext": false
			},
			
			// Deshabilitar autocompletado nativo en LaTeX
			"[latex]": {
				"editor.quickSuggestions": {
					"comments": "off",
					"strings": "off",
					"other": "off"
				},
				"editor.suggestOnTriggerCharacters": false,
				"editor.tabCompletion": "off",
				"editor.formatOnSave": true
			},
			
			"editor.inlineSuggest.enabled": true
		}
	\end{lstlisting}
	
	\subsection{Crear archivo .gitignore}
	
	Crear un archivo \codigo{.gitignore} en la raíz del proyecto:
	
	\begin{lstlisting}[caption=.gitignore]
		# Archivos temporales de LaTeX
		*.aux
		*.log
		*.out
		*.toc
		*.synctex.gz
		*.fls
		*.fdb_latexmk
		
		# Directorio de compilación
		/build/
		/out/
		
		# Archivos de sistema
		.DS_Store
		Thumbs.db
		desktop.ini
	\end{lstlisting}
	
\tip{El archivo \codigo{.gitignore} evita que subas archivos temporales innecesarios a GitHub.}

	\section{Uso Práctico de GitHub Copilot}
	
	\subsection{¿Cómo funciona?}
	
	Copilot analiza el contexto de tu documento (comentarios, código previo, estructura del proyecto) y sugiere el siguiente fragmento de código. Para aceptar una sugerencia, simplemente presiona \comando{Tab}.
	
	\begin{figure}[H]
		\centering
		\fcolorbox{azulprofundo}{grisclaro}{
			\begin{minipage}{0.85\textwidth}
				\centering
				\vspace{0.5cm}
				\textcolor{azulprofundo}{\textbf{Ejemplo visual}}
				\vspace{0.2cm}
				
				\codigo{\% Crear una ecuación de la Ley de Ohm} \\
				\vspace{0.2cm}
				\textcolor{gray}{[Copilot sugiere:]}
				
				\textbf{\textcolor{azulprofundo}{\textbackslash begin\{equation\}}} \\
				\textbf{\textcolor{azulprofundo}{\quad V = I \textbackslash cdot R}} \\
				\textbf{\textcolor{azulprofundo}{\textbackslash end\{equation\}}}
				\vspace{0.5cm}
			\end{minipage}
		}
		\caption{Ejemplo de autocompletado con Copilot}
		\label{fig:ejemplo}
	\end{figure}
	
	\subsection{Generación de Estructura de Documento}
	
	\textbf{Escribe el siguiente comentario:}
	
	\begin{lstlisting}
		% Crear la estructura de un informe de laboratorio de electrónica con:
		% - Portada con título, autor, fecha
		% - Índice automático
		% - Secciones: Introduccion, Marco Teorico, Metodologia, Resultados, Conclusiones
	\end{lstlisting}
	
	\noindent \textbf{Copilot generará automáticamente} el código LaTeX completo con toda la estructura.

	\section{Solución de Problemas Comunes}
	
	\begin{longtable}{|p{0.35\textwidth}|p{0.6\textwidth}|}
		\hline
		\textbf{Problema} & \textbf{Solución} \\
		\hline
		Copilot no sugiere nada & Verificar que la extensión está habilitada (ícono en barra inferior debe mostrar \faCheck) \\
		\hline
		Error de compilación LaTeX & Verificar que MiKTeX está instalado y en PATH. Abrir terminal y ejecutar \codigo{pdflatex --version} \\
		\hline
		PDF no se actualiza & Cerrar y reabrir el PDF en VS Code, o limpiar archivos temporales con \comando{Ctrl+Shift+P} → \codigo{LaTeX Workshop: Clean up} \\
		\hline
		Copilot no se autentica & \comando{Ctrl+Shift+P} → \codigo{GitHub Copilot: Sign in} \\
		\hline
		Conflictos con autocompletado & Asegurar que la sección \codigo{[latex]} está configurada en \codigo{settings.json} \\
		\hline
		Error de espacio en decimales & Revisar código TikZ generado. Reemplazar \codigo{(3. 5cm)} por \codigo{(3.5cm)} \\
		\hline
		La compilación es muy lenta & Instalar \codigo{latexmk} adicional. En MiKTeX Console → Packages → buscar "latexmk" → Install \\
		\hline
		No se ven las imágenes & Verificar que la ruta de las imágenes es correcta. Usar rutas relativas: \codigo{\textbackslash includegraphics\{figuras/imagen.png\}} \\
