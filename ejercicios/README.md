# Ejercicios del Curso

## Módulo 1: Fundamentos de LaTeX

### Ejercicio 1: Ecuaciones Matemáticas
**Archivo:** `ejercicio1_ecuaciones.tex`

Crear un documento que contenga:
- 5 ecuaciones de electrónica (Ley de Ohm, Potencia, Reactancia, Impedancia, Frecuencia de resonancia)
- Ecuaciones numeradas
- Referencias cruzadas

### Ejercicio 2: Tablas y Datos
**Archivo:** `ejercicio2_tablas.tex`

Crear una tabla con:
- 6 mediciones de voltaje vs corriente
- Título y etiqueta
- Formato profesional

### Ejercicio 3: Informe Completo
**Archivo:** `ejercicio3_informe.tex`

Elaborar informe completo sobre "Circuito RC" que incluya:
- Portada, índice, secciones
- Ecuaciones, tabla, figura
- Bibliografía

## Módulo 2: LaTeX + IA

### Ejercicio 4: Corrección Asistida
**Archivo:** `ejercicio4_correccion.tex`

Usar ChatGPT para:
- Identificar errores en código dado
- Sugerir mejoras
- Documentar los prompts utilizados

### Ejercicio 5: Proyecto Integrador
**Archivo:** `ejercicio5_proyecto.tex`

Desarrollar documento completo (mínimo 3 páginas) usando IA en todas las etapas.

### Ejercicio 6: Mathpix
**Archivo:** `ejercicio6_mathpix.tex`

Convertir imagen de ecuación a LaTeX usando Mathpix.

### Ejercicio 7: Comparación
**Archivo:** `ejercicio7_comparacion.tex`

Comparar resultados de 2 herramientas de IA diferentes.

### Ejercicio 8: Artículo Avanzado
**Archivo:** `ejercicio8_avanzado.tex`

Elaborar artículo estilo IEEE (4-5 páginas) con apoyo de IA.
